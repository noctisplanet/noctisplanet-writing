隐私政策

最后更新: 2026-07-07

磨译 Writing ("本 App") 是一款面向中文母语者的英语语法练习应用. 本政策说明你的 数据如何被处理. 一句话: 本 App 没有我们的服务端, 不收集任何数据, 所有内容都保存 在你的设备上.

没有开发者服务端, 不收集数据

磨译的开发者不为本 App 运营任何后端服务器. 我们不收集, 不接收, 不存储, 不传输, 也无法访问你的个人数据, 练习内容, 成绩或 API 密钥. 本 App 无需账号, 无需登录, 不含 任何分析, 广告或追踪 SDK.

你的数据存放在哪里

  • 试卷, 题目, 作答, 成绩与进度都保存在你的设备本地.
  • 你的大模型 API 密钥经过加密后保存在你的设备本地.
  • 以上内容都不会发送给开发者.

第三方大模型服务商 (自备密钥)

本 App 本身不包含任何 AI 模型或 API 密钥. 出题, 批改与句子成分分析需要你自行配置 第三方大模型服务商的 API 密钥 (例如 DeepSeek 或通义千问 Qwen).

使用这些功能时, 本 App 会把相关内容 - 中文题干和你的英文作答 - 从你的设备直接发送 给你所选择的服务商, 并用你自己的 API 密钥进行鉴权. 该请求不会经过任何由我们运营的 服务器.

这些内容随后由第三方服务商按其自身的隐私政策与条款处理. 我们无法控制, 也不对这些 服务商如何处理你的数据负责. 请查阅你所使用服务商的政策, 例如:

  • DeepSeek: https://www.deepseek.com
  • 通义千问 Qwen (阿里云百炼): https://www.aliyun.com

用量费用由你所选服务商直接向你收取. 本 App 不处理任何支付, 也不收取任何佣金.

朗读发音

朗读功能使用设备操作系统内置的语音合成, 不会把任何文本发送给我们.

儿童隐私

本 App 不收集任何数据, 也并非专门面向儿童. 使用 AI 功能需要第三方 API 密钥与账号, 它们有各自的年龄与资格要求.

删除你的数据

由于所有数据都保存在你的设备上, 你可以通过删除本 App, 或在 App 内清除单份试卷与已 保存的密钥, 来移除全部数据.

政策变更

若本政策有变更, 更新后的版本将发布在本 URL, 并标注新的 "最后更新" 日期.
