常见问题

无法出题 / 点批改没反应.
磨译不含内置 AI 模型, 需要你先在设置里添加自己的大模型 API 密钥. 打开 设置 -> 某个 服务商 (DeepSeek 或通义千问 Qwen) -> 用你的 API 密钥登录. 没有有效密钥, AI 功能无法 运行.

支持哪些服务商?
DeepSeek 与通义千问 Qwen (OpenAI 兼容的对话补全 API).

密钥存在哪? 我的数据会被上传吗?
密钥经过加密后保存在你的设备本地, 试卷也一样. 本 App 没有我们的服务端; 你的内容用 你自己的密钥直接发送给你所选的大模型服务商. 详见隐私政策.

如何计费?
App 为一次性买断. API 用量由你所选服务商直接计费, App 不收取任何佣金.
